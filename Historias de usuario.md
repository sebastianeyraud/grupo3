# Historias de Usuario - Grupo A, Equipo 3  
**Proyecto: Manuales Frecuentes**

---

## 1) Descargar Manuales

**Historia de Usuario:**  
*Como cliente, quiero en la página web una opción para descargar manuales.*

**Requerimientos:**
- El sistema debe mostrar un botón para poder descargar.
- El sistema debe tener una opción para seleccionar formato y calidad.
- El sistema debe guardar un historial de descargas.

---

## 2) Visualizar Manuales

**Historia de Usuario:**  
*Como cliente, quiero poder previsualizar el contenido del manual antes de descargarlo para asegurarme de que es el que necesito.*

**Requerimientos:**
- El sistema debe mostrar una vista previa del contenido a descargar.
- El sistema debe tener un apartado para poder hacer vistas previas.
- El sistema no debe descargar los manuales automáticamente.

---

## 3) Crear Cuenta

**Historia de Usuario:**  
*Como nuevo usuario, quiero poder crear una cuenta para usar todas las funciones del sitio.*

**Requerimientos:**
- El sistema debe tener un botón para ingresar a la pantalla de registro.
- El sistema debe tener una pestaña para ingresar los datos (nombre, apellido, RUT, e-mail, contraseña).
- El sistema debe poder validar cada dato de registro.
- El sistema debe tener opción para crear cuenta.
- El sistema debe tener opción para cancelar cuenta.

---

## 4) Login (Iniciar Sesión)

**Historia de Usuario:**  
*Como cliente, quiero iniciar sesión con nombre y contraseña para mantener mi información personal segura.*

**Requerimientos:**
- El sistema deberá pedir correo y contraseña para iniciar sesión.
- El sistema debe pedir registro si el usuario no tiene una cuenta.
- El sistema debe verificar si los datos son correctos.
- El sistema debe permitir iniciar sesión al cliente con sus datos ingresados correctamente.

---

## 5) Modificar Perfil

**Historia de Usuario:**  
*Como usuario registrado, quiero editar mi perfil para mantener mi información actualizada.*

**Requerimientos:**
- El sistema debe poder modificar perfil sólo con cuenta logueada.
- El sistema debe tener un botón para modificar perfil dentro de las opciones de cuenta.
- El sistema debe mostrar una pestaña para ingresar los datos (nombre, RUT, e-mail, contraseña).
- El sistema debe poder validar cada dato de registro.
- El sistema debe tener opción para confirmar o cancelar la modificación.

---

## 6) Comentarios

**Historia de Usuario:**  
*Como cliente, quiero poder dejar comentarios en los manuales para compartir mi opinión o experiencia con otros usuarios.*

**Requerimientos:**
- El sistema debe tener un apartado para realizar comentarios en las publicaciones de los manuales.
- El sistema debe permitir comentarios con 2000 caracteres máximo.

---

## 7) Videos

**Historia de Usuario:**  
*Como cliente, quiero ver videos relacionados con los manuales para comprender mejor cómo usar el producto.*

**Requerimientos:**
- El sistema debe manejar la carga de videos mientras el usuario los utiliza.
- El sistema debe mostrar el video en la calidad original.
- El sistema debe poder pausar y silenciar el video.

---

## 8) Valoración

**Historia de Usuario:**  
*Como cliente, quiero calificar los manuales (de 0 a 5 estrellas) para ayudar a otros usuarios a saber cuáles son útiles.*

**Requerimientos:**
- El sistema debe tener un apartado para calificar los manuales.
- La valoración debe ser con estrellas, siendo 0 la mínima y 5 la máxima calificación.

---

## 9) Sistema de Guardado

**Historia de Usuario:**  
*Como usuario registrado, quiero guardar manuales en favoritos para acceder a ellos rápidamente más tarde.*

**Requerimientos:**
- El sistema debe tener una opción para marcar un manual como favorito.
- El sistema debe tener una lista con los manuales favoritos del usuario.

---

## 10) Buscador

**Historia de Usuario:**  
*Como cliente, quiero buscar manuales por nombre para encontrarlos fácilmente.*

**Requerimientos:**
- El sistema debe tener un buscador incorporado.
- El buscador debe funcionar con palabras clave.

---

## 11) Filtro de Búsqueda

**Historia de Usuario:**  
*Como cliente, quiero aplicar filtros en la búsqueda para refinar los resultados según mis necesidades.*

**Requerimientos:**
- El sistema debe tener filtros por:
  - Nombre
  - Autor
  - Tipo de electrodoméstico

---

## 12) Soporte Técnico

**Historia de Usuario:**  
*Como cliente, quiero acceder a soporte técnico para recibir ayuda en caso de problemas con la página o los manuales.*

**Requerimientos:**
- La página debe tener un apartado para soporte técnico.
- El soporte debe estar disponible 24/7.
- El sistema debe brindar al soporte la información necesaria para resolver las dudas.

---

## 13) Subir Manuales

**Historia de Usuario:**  
*Como usuario registrado, quiero subir manuales a la plataforma para compartir información con otros usuarios.*

**Requerimientos:**
- La página debe tener una opción para subir manuales desde el perfil.
- El tamaño máximo del manual debe ser 10 MB.
- El usuario no tiene límite en la cantidad de manuales que puede subir.
- El usuario debe estar registrado.

---

## 14) Ficha Técnica

**Historia de Usuario:**  
*Como cliente, quiero ver la ficha técnica de los productos para conocer sus especificaciones detalladas.*

**Requerimientos:**
- El sistema debe tener una opción para ver la ficha técnica antes del manual.
- Se debe verificar que la ficha técnica sea correspondiente al equipo.

---

## 15) Compatibilidad con Navegador

**Historia de Usuario:**  
*Como cliente, quiero que el sistema sea compatible con distintos navegadores para acceder desde cualquier dispositivo.*

**Requerimientos:**
- El sistema debe ser compatible con Google Chrome, Mozilla Firefox, Microsoft Edge, etc.

---

## 16) Recuperar Contraseña

**Historia de Usuario:**  
*Como usuario, quiero recuperar mi contraseña si la olvido para no perder acceso a mi cuenta.*

**Requerimientos:**
- El sistema debe tener una opción para recuperar la cuenta.
- El sistema debe enviar un correo con un código de activación al correo vinculado.

---

## 17) Editar Manuales Subidos

**Historia de Usuario:**  
*Como usuario registrado, quiero modificar los manuales que he subido para corregir errores o actualizarlos.*

**Requerimientos:**
- El sistema debe mostrar un botón de editar cerca de los manuales subidos.
- Se debe poder modificar el PDF, ficha técnica y videos del usuario.
- Debe haber opción para cancelar o aplicar la modificación.
- El sistema debe reflejar las modificaciones realizadas.

---

## 18) Diseño Grid System

**Historia de Usuario:**  
*Como cliente, quiero una interfaz basada en un sistema de diseño grid para visualizar el contenido de forma ordenada y clara.*

**Requerimientos:**
- El sistema debe estar diseñado con bloques tipo grid, VBox y HBox.
- Las funciones principales deben estar agrupadas de forma visual clara.

---

## 19) Modificar Comentario

**Historia de Usuario:**  
*Como usuario, quiero editar mis comentarios para corregir errores o mejorar mi aporte.*

**Requerimientos:**
- El sistema debe tener una opción para editar un comentario ya publicado.
- El sistema debe permitir cancelar la edición antes de aplicarla.
- El sistema debe mostrar el comentario modificado.

---

## 20) Modificar Valoración

**Historia de Usuario:**  
*Como usuario, quiero cambiar mi valoración de un manual para reflejar mejor mi experiencia después de usarlo.*

**Requerimientos:**
- El sistema debe permitir modificar una valoración anterior.
- Se debe poder cancelar la nueva valoración antes de aplicarla.
- El sistema debe mostrar la nueva valoración modificada.
